# Mix Platform 🚀

**Mix Platform** هو نظام متكامل يدمج بين إدارة الطاقة، الخلايا، الألعاب، الشبكات الاجتماعية، وعالم الأحلام ثلاثي الأبعاد، مع دعم Metaverse وThree.js.  
تم تصميمه كنظام تشغيل شخصي/تجريبي لتحليل الطاقة، التفاعل الاجتماعي، والأحلام الرقمية.

---

## 🌟 المميزات الرئيسية

- 🔐 **نظام تسجيل وحماية JWT** للمستخدمين.
- 📊 **Dashboard شخصي** لعرض الطاقة، الترددات، والخلايا.
- 🧬 **نظام الخلايا** لكل عملية أو نشاط.
- ⏳ **تطور الطاقة تلقائيًا** عبر Scheduler.
- 🌐 **Metaverse + Three.js** لإنشاء عوالم تفاعلية.
- 🌙 **Dream Engine** لتحليل الأحلام وإنشاء عالم الأحلام.
- 🎮 **ألعاب مدمجة** (مثل لعبة BTC Game).
- 💬 **شبكات اجتماعية** مع نشر منشورات، تعليقات، ورسائل.

---

## 🗂 هيكل المشروع

```text
Mix_platform/
│
├── backend/                       # النظام الخلفي
│   ├── core/                       # الوظائف الأساسية
│   ├── api/                        # واجهات API لكل وحدة
│   ├── modules/                    # موديولات منفصلة (AI, Wallet, Games, Social)
│   ├── models/                     # نماذج البيانات
│   ├── routes/                     # مسارات REST API
│   └── utils/                      # أدوات مساعدة
│
├── frontend/                       # الواجهة + Three.js
│   ├── static/                      # سكربتات JS و Three.js
│   ├── *.html                        # صفحات الويب
│   └── style.css                     # تصميم الواجهة
│
├── assets/                          # الموارد (3D Models, Textures, Audio)
├── storage/                         # ملفات JSON للتخزين المؤقت وقاعدة بيانات بسيطة
├── scripts/                         # سكربتات التشغيل، النشر، النسخ الاحتياطي
└── logs/                            # سجلات النظام
